# HastaKalaShop 🎨📱

A modern Android application developed using **Kotlin + Jetpack Compose** for helping small artisans and micro-business owners manage sales, inventory, and product analytics intelligently.

Built as part of the **MindMatrix VTU Internship Program** under the project:

**Android App Development using GenAI – Hasta-Kala Shop (Self-Employment)**

---

## 📌 Problem Statement

Small craft artisans selling handmade products such as:
- Banana fiber bags
- Keychains
- Handmade accessories

often struggle with:
- Inventory tracking
- Identifying best-selling products
- Avoiding dead stock
- Managing sales data manually

HastaKalaShop solves this by providing a lightweight sales analytics and billing application.

---

# ✨ Features

## 🧾 Quick Billing
- Fast product entry
- Select item color/design
- Save sales instantly

## 📊 Sales Analytics Dashboard
- Pie charts & bar charts
- Best-selling product insights
- Weekly/Monthly sales overview

## 🚨 Stock Alerts
- Low stock notifications
- Helps artisans know what to produce more

## 💰 Income Tracking
- Track total earnings
- Filter by:
  - This Week
  - This Month

## 🎨 Modern UI
- Built using Jetpack Compose
- Clean and intuitive design
- Mobile-friendly workflow
